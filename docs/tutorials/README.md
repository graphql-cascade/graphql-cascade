# Tutorials

This directory contains step-by-step tutorials for implementing GraphQL Cascade in real applications.

## Available Tutorials

- **[Todo App](todo-app.md)** - Build a complete CRUD todo application with automatic cache updates
- **[Blog Platform](blog-platform.md)** - Complex relationships and real-time features
- **[Real-time Collaboration](real-time-collab.md)** - Multi-user editing with conflict resolution

## Learning Path

1. **Start Here**: [Todo App](todo-app.md) - Learn the basics with a simple CRUD application
2. **Next Steps**: [Blog Platform](blog-platform.md) - Handle complex entity relationships
3. **Advanced**: [Real-time Collaboration](real-time-collab.md) - Real-time features and conflict resolution

## Prerequisites

Before starting these tutorials, make sure you have:

- Basic knowledge of GraphQL
- Familiarity with your chosen client library (Apollo, React Query, etc.)
- Node.js and npm installed for client development
- Python or Node.js for server development
